Can you show me how to implement a basic RAG retrieval function using Python and ChromaDB? I need to handle multi-document context.

To implement a multi-document retrieval system using ChromaDB, you'll need to initialize a persistent client, create a collection with your embedding function, and then query based on similarity. Here is a streamlined example using the latest ChromaDB SDK:

Python 
import chromadb
from chromadb.utils import embedding_functions

# Initialize client
client = chromadb.PersistentClient(path="./db")

# Use OpenAI or SentenceTransformers for embeddings
emb_fn = embedding_functions.SentenceTransformerEmbeddingFunction()

collection = client.get_or_create_collection(
    name="documentation_store", 
    embedding_function=emb_fn
)

def query_rag(prompt, n_results=3):
    results = collection.query(
        query_texts=[prompt],
        n_results=n_results
    )
    return results['documents'][0]
